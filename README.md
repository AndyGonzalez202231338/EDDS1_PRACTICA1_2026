# JUEGO UNO - Versión en C++

## Descripción
Implementación del clásico juego de cartas UNO en C++ con modo Clásico y modo FLIP.  
El juego utiliza listas enlazadas para representar las manos de los jugadores y el mazo.

---

## Requisitos del Sistema

### Software necesario:
- **Sistema operativo:** Linux (Ubuntu 18.04+, Debian 10+, o cualquier distribución)
- **Compilador:** g++ (versión 7.0 o superior)
- **Make:** versión 4.0 o superior
- **Terminal:** Cualquier terminal con soporte para colores ANSI

### Verificar instalaciones:
```bash
g++ --version
make --version


# Ubuntu/Debian
sudo apt update
sudo apt install g++ make

# Fedora
sudo dnf install gcc-c++ make

# Arch Linux
sudo pacman -S gcc make
